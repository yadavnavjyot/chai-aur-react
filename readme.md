# chai-aur-react

A minimal starter project combining React with Chai-based testing. "Chai aur React" demonstrates a simple React app setup, development scripts, and a testing setup using Mocha + Chai.

## Features

- React app scaffold (Vite / Create React App compatible)
- Example component and test written with Mocha + Chai
- npm scripts for install, start, build, and test

## Prerequisites

- Node.js (v14+ recommended)
- npm or yarn

## Installation

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm start
```

## Testing

Run unit tests (Mocha + Chai):

```bash
npm test
```

## Project Structure

- `src/` — React components and app code
- `test/` — Mocha/Chai tests
- `public/` — Static assets
